# Resume Library (Versioned)

This folder contains role-targeted resume variants for Brian Hannigan.

## Structure
- `general/`: master resume (broad)
- `python-backend/`: Python backend engineer variant
- `cybersecurity/`: SIEM / Detection / GRC variant
- `ai-architect/`: AI systems architect variant

## Generation
Files are generated by `generate_resumes.py` using `python-docx`.
TXT versions are included for ATS checks and easy copy/paste.
